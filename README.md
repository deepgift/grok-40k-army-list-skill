# Grok 40k List Skill

**A specialized Grok skill that turns Grok into a tournament-level, data-driven analyst for Warhammer 40,000 10th Edition army building.**

This is one of the first public Grok skills dedicated to competitive 40k list construction. It provides objective, sourced analysis with no army bias, focusing on points efficiency, synergies, meta positioning, and real tournament results.

---

## ✨ What This Skill Does

- Analyzes any unit, combo, or detachment with hard data (Wahapedia rules + Goonhammer + 2026 GT results)
- Builds balanced 2000-point list skeletons with clear reasoning
- Gives honest pros/cons with sources for every recommendation
- Maintains strict neutrality across all factions
- Follows a consistent, professional output structure for easy reading

**Perfect for:**
- Competitive players building new lists
- People wanting meta-aware advice without hype
- Theory-crafting around specific centerpieces
- Understanding why certain units perform (or fail) in the current meta

---

## 📥 How to Install & Use

### Option 1: Quick Install (Recommended)
1. Copy the entire content of `SKILL.md`
2. Go to Grok (grok.com or the app)
3. Start a new conversation and say:
   > "Load this as a permanent skill: [paste the entire SKILL.md content]"
4. Grok will confirm it's saved as a reusable skill.

### Option 2: File Upload
1. Download `SKILL.md` from this repo
2. In Grok, go to **Skills** → **Upload new skill** (or ask Grok to load a file)
3. Upload the file

Once loaded, just start any conversation with 40k questions and the skill will activate automatically when relevant.

**Example prompts that trigger it well:**
- "Build a competitive list around the new [unit name]"
- "Is the [detachment] good in the current meta?"
- "Analyze the efficiency of [high-point unit] for 2000 points"
- "Help me build a [faction] list for a GT this weekend"
